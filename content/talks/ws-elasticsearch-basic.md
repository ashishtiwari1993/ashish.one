# Sample Queries for Elasticsearch Workshop

## CRUD 

```

# Insert

POST meetup/_doc/
{
  "name":"Ashish Tiwari"
}

# Insert with id

POST meetup/_doc/1
{
  "name":"Ashish Tiwari"
}

# Search

GET meetup/_search

# Update

POST meetup/_doc/1
{
  "name":"Ashish",
  "company":"elastic",
  "address":"Navi Mumbai kharghar",
  "skills":{
    "language":["php","java","node"],
    "database":["mysql","mongodb"],
    "search":"elasticsearch"
  }
}

# search with query

GET meetup/_search
{
  "query": {
    "match": {
      "address": "navi"
    }
  }
}

# delete

DELETE meetup
```
